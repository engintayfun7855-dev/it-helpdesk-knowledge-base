# IT Helpdesk Knowledge Base

This repository contains ticket-based troubleshooting documentation for common IT helpdesk and desktop support scenarios.

The goal of this project is to demonstrate structured problem-solving, user support workflow, basic Windows troubleshooting, network diagnostics, printer support, Outlook issue handling, and Active Directory account support.

## Topics Covered

- Windows troubleshooting
- Network connectivity issues
- Printer troubleshooting
- Microsoft Outlook support
- User account and password reset scenarios
- Basic Active Directory support workflow
- Helpdesk ticket documentation

## Ticket Scenarios

| Ticket | Topic | Category |
|---|---|---|
| Ticket 001 | User cannot connect to the internet | Network Support |
| Ticket 002 | Printer is not printing | Printer Support |
| Ticket 003 | Outlook is not opening or syncing | Microsoft Office Support |
| Ticket 004 | Computer is running slow | Windows Performance |
| Ticket 005 | User forgot password | Active Directory / Account Support |

## Purpose

This project was created as a practical IT support portfolio project for entry-level Helpdesk, Desktop Support, and IT Support roles.

It focuses on documenting real-world support scenarios using a clear troubleshooting process:

1. Understand the user problem
2. Ask the right questions
3. Check basic causes first
4. Use relevant diagnostic commands
5. Identify the root cause
6. Apply the solution
7. Document the result
